# Professional Software Engineer Portfolio

This is a complete, professional portfolio website designed for a Software Engineer.

## Features
- **Modern Dark Theme**: Professional and sleek design.
- **Responsive Layout**: Works perfectly on mobile, tablet, and desktop.
- **Interactive Elements**: Typing text effect, scroll animations, and dynamic skill bars.
- **Sections Included**:
  - Hero (Introduction)
  - About Me
  - Technical Skills
  - Featured Projects
  - Work Experience
  - Contact Form

## How to Use
1. **Open the Website**: Simply double-click `index.html` to view it in your browser.
2. **Customize Content**:
   - Open `index.html` in a text editor (like VS Code or Notepad).
   - Replace "SE Priyankara" with your name.
   - Update the "Experience" and "Projects" sections with your own details.
   - Update the `mailto:` email address in the Contact section.

## Files
- `index.html`: The main structure of the website.
- `styles.css`: All the styling and visual design.
- `script.js`: Interactive features and animations.

## Customization
You can easily change the color scheme by modifying the variables at the top of `styles.css`.
